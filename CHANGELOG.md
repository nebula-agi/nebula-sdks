# Changelog

All notable changes to the Nebula SDKs will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## JavaScript SDK

### [js-v0.0.33] - 2024-11-17

#### Added
- Initial open source release
- Complete TypeScript SDK for Nebula API
- Memory management (create, update, delete)
- Collection management
- Vector search capabilities
- Graph query support
- Full async/await support
- Comprehensive error handling

#### Changed
- Migrated to open source monorepo

## Python SDK

### [py-v0.1.8] - 2024-11-17

#### Added
- Initial open source release
- Complete Python SDK for Nebula API
- Sync and async client implementations
- Memory management (create, update, delete)
- Collection management
- Vector search capabilities
- Graph query support
- Complete type hints
- Comprehensive error handling

#### Changed
- Migrated to open source monorepo
- Updated license from Proprietary to MIT
